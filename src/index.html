Let me introduce myself

Since creating my first code snippet in JavaScript I have quickly discovered a strong passion towards problem solving and creating web applications. My ultimate goal is to become a MERN master, to make my code reusable, elegant, well-optimized and easy to work with.

Core principles of my projects: optimization, minimalism, positive user experience.
